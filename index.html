Crystal Collector - A Mathmatical Journey

How to play:

A random number will appear at the start of the game.

Each of the four crystals below will be given a specific number for the duration of the game.

However, the value of each crystal will not be revealed until you have clicked it.

The goal is to match the randomly selected number by strategically clicking the crystals.

To win, do the math, choose wisely and match the number. Go over, and you lose.

With every game start, the fun begins anew. A new random number and new values to each crystal.
